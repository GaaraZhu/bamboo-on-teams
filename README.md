# BAMBOO-ON-TEAMS #
A serverless ChatOps tool for interacting with Bamboo from Microsoft Teams

# SYNOPSIS #
\<command\> [options]

Use "<command> help" for information on a specific command. The synopsis for each command shows its options and their usage.

# AVAILABLE COMMANDS #
## Build commands ##
* list-plans
* list-branches
* list-builds
* desc-build
* create-branch
* build

## Deploy commands ##
* list-projects
* list-envs
* list-releases
* list-deploys
* create-release
* deploy-latest
* deploy-release
* deploy-build
* promote-release
